```bash
conda create -n my_project python=3.8
```

생성 완료되면 다음과 같은 명령어 보여준다

```bash
# To activate this environment, use
#
#     $ conda activate my_project
#
# To deactivate an active environment, use
#
#     $ conda deactivate
```
